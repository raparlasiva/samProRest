Modules (File Name) : 1. The name of the Module should start with a small letter for the first word and a capital letter for every other word ex:- inventoryItems Controllers (Inside PHP Class): 1. The name of the Controller should start with a Capital letter for the first word and a capital letter with every other word. It should have a controller at the end of the word. No plural names in the controller name ex:- class InventoryItemController and not class InventoryItemsController Controllers (File Name): 1. The whole name of name of the Controller should be in small letters and have a controller at the end of the word. No plural names in the controller name ex:- inventoryitemcontroller and not inventoryitemscontroller Models (Inside PHP Class): 1. The name of the Controller should start with a Capital letter for the first word and a capital letter with every other word. It should have a controller at the end of the word. No plural names in the controller name. ex:- class InventoryItemModel and not class InventoryItemsModel Models (File Name): 1. The whole name of name of the models should be in small letters and have a model at the end of the word. No plural names in the model name ex:- inventoryitemmodel and not inventoryitemsmodel Javascript (File Name) :- 1. The javascript file name should be same as the module name. ex:- inventoryItemsModule.js
